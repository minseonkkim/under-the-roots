# 05. 12주 MVP 로드맵

> 솔로 개발자 기준, 주당 20~30시간 투입 가정. **아트·사운드는 최대한 외부 에셋 활용**, 핵심 시스템에 시간 집중.

## 5.1 마일스톤 개요

| 주차 | 마일스톤 | 주요 산출물 |
| --- | --- | --- |
| 1~2 | **프로토타입** | 한 캐릭터, 1개 방, 자동 전투, 잡몹 1종 |
| 3~4 | **핵심 루프** | 5층 던전 1개, 룬 시스템, 보스 1개 |
| 5~6 | **메타 시스템** | 마을, 도토리 나무, 캐릭터 4명, 세이브 |
| 7~8 | **콘텐츠 확장** | 캐릭터 8명, 룬 32개, 잡몹 12종 완료 |
| 9 | **수익화 통합** | AdMob, Google Play Billing, 분석 |
| 10 | **밸런스·튜닝** | 데이터 수치 패스, UX 패스 |
| 11 | **클로즈 베타** | 친구·인디카페 5~10명 베타. 버그 수정 |
| 12 | **스토어 등록 & 출시** | 스크린샷, 영상, 사전등록, 런칭 |

## 5.3 컷 라인 (Scope Cut Priority)

일정이 밀리면 **위에서부터 잘라낸다**.

1. ~~로컬라이제이션 영어~~ → 한국어만 출시, 영어는 +2주 후
2. ~~데일리 미션~~ → 출시 후 1차 업데이트
3. ~~진화 룬 일부~~ → 6종 → 3종으로 축소
4. ~~캐릭터 8명 → 5명~~
5. ~~사운드 직접 제작~~ → 무료 에셋 100% 활용
6. ~~Crashlytics~~ → Firebase 기본만, 구글 기본 충돌 리포트 의존

이 6개를 모두 잘라내면 12주 → 8주로 단축 가능.

## 5.6 출시 성공 기준 (Definition of Success)

**MVP**의 목표는 매출이 아닌 **검증**.

| 기간 | 검증 목표 |
| --- | --- |
| 출시 1주 | D1 잔존율 30% 이상 → "재미는 있다" |
| 출시 2주 | 평균 세션 8분 이상 → "코어 루프 작동" |
| 출시 1개월 | 광고 시청률 40% 이상, IAP 1% 이상 → "수익 가능성" |
| 출시 3개월 | 월 매출 100만원 이상 → "지속 개발 가능" |

위 4개 중 3개 이상 달성 시 **던전 2~4 풀 콘텐츠 개발**로 진행. 미달 시 **피벗 또는 다음 프로젝트** 전환 결정.
